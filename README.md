# Todo App Challenge
Todo app built with React Native

## Getting Started

### Prerequisites
MongoDB
```
$ npm install mongodb --save
```

### Installing
- clone or repo
- cd into project folder

```
$ npm install
```

### Running Project

```
$ npm run start
```

once finished, in a separate terminal window

```
$ react-native run-ios
or
$ react-native run-android
```

start mongoDB

```
$ mongod
```

then start server

```
$ npm run server
```
